# Description

Very basic flexbox grid demo. Resize your viewport to see how it responds to
width changes.

Back to [Layout#Grid](layout/#grid).

```html
<div class="grid-2 md-grid-4 lg-grid-6">
  <div>One</div>
  <div>...</div>
  <div>Twenty</div>
</div>
```
