# Description

Very basic flexbox grid demo. Resize your viewport to see how it responds to
width changes. The classes are cosmetic.

Back to [Components#sf-grid](components/#sf-grid).

```html
<sf-grid class="doc-grid-demo pad-ch">
  <p>One</p>
  <p>Two</p>
  <p>Three</p>
  <p>Four</p>
  <p>Five</p>
  <p>Six</p>
  <p>Seven</p>
  <p>Eight</p>
  <p>Nine</p>
  <p>Ten</p>
</sf-grid>
```
